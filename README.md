# TP Rehabilitación

## Descripción
El trabajo consiste en un menú desde el cual se puede acceder a 3 juegos en los que se evalúa la exploración espacial, atención y tiempos de registro. 
Los resultados se guardan en un archivo json en la misma carpeta donde se encuentra el código del programa. Se puede buscar por el número de ID del paciente (este se ingresa al principio, antes del menú).


---

## Tecnologías utilizadas

* Python
* Librerías: pygame, random, time, os, json

---

## Instalación

1. Clonar el repositorio

```bash
git clone <https://github.com/valjeon/tp-rehabilitacion>
cd <tp-rehabilitacion>
```

2. Crear entorno virtual

```bash
python -m venv venv
```

3. Activar entorno (Windows)

```bash
venv\Scripts\activate
```

4. Instalar dependencias

```bash
pip install -r requirements.txt
```

---

## ▶️ Uso

Ejecutar el programa principal:

```bash
python menuRehab.py
```
